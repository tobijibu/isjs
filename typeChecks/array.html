array()

更新日: 2018.09.14

is.array(value:any)

渡された値がarrayかどうかを判定します。

インターフェース:
not, all, any
is.array(['foo', 'bar', 'baz']);
=> true

is.not.array({foo: 'bar'});
=> true

is.all.array(['foo'], 'bar');
=> false

is.any.array(['foo'], 'bar');
=> true

// 'all'と'any'は、配列を受け取ることができます。
is.all.array([[1, 2], 'foo', 'bar']);
=> false

デモ

変数を判定してconsole.logを出力します。

JavaScript
var val = ['foo', 'bar', 'baz'];
if (is.array(val)) {
  console.log("['foo', 'bar', 'baz']は配列です。");
}
