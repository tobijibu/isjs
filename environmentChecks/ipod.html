ipod()

作成日: 2018.05.18

is.ipod(range:number|string)

現在の端末がiPodかどうかを判定します。
rangeはオプションです。iOSバージョンまたは、iOSバージョンの範囲を指定します。

インターフェース:
not
is.ipod();
=> 端末がiPodならtrueを返します。
 
is.not.ipod();
=> 端末がiPodでないならtrueを返します。
 
// iOSバージョン番号もサポートしています。
is.ipod(7);
=> 端末がiPodでiOSバージョン7ならtrueを返します。
 
is.ipod('>=5');
=> 端末がiPodでiOSバージョン5以上ならtrueを返します。
 
is.not.ipod('<5');
=> 端末がiPodでiOSバージョン5未満でなければtrueを返します。

デモ

端末を判定してconsole.logを出力します。

JavaScript

if (is.ipod()) {
  console.log('使用している端末はiPodです。');
} else {
  console.log('使用している端末はiPodではありません。');
}
