Introduction

更新日: 2018.09.14

Introduction

is.jsは汎用チェックライブラリです。

  • 依存関係はありません。
  • AMDや、Node、ブラウザで利用可能です。

Usage

Node.js

$ npm install is_js

Bower

$ bower install is_js

ビルド

$ npm run build

テスト

$ npm test
